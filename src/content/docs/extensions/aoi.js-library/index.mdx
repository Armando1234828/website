A plugin library for aoi.js that adds more functions to your aoi.js bot.

### Features

#### Create your own plugins

- aoi.js-library offers the ability to create and publish your own aoi.js related plugins for every user to use!

---

## Setup

To get started with aoi.js-library, follow these steps:

    **node.js 16.9.0 or newer is required.**


    ```bash
    npm install aoi.js-library
    ```

---

```javascript title="index.js"
const { AoiClient } = require("aoi.js");
const { PluginManager } = require("aoi.js-library");

const client = new AoiClient({
  token: "DISCORD BOT TOKEN",
  prefix: "DISCORD BOT PREFIX",
  intents: ["MessageContent", "Guilds", "GuildMessages"],
  events: ["onMessage"],
  database: {
    type: "aoi.db",
    db: require("@akarui/aoi.db"),
    tables: ["main"],
    path: "./database/",
    extraOptions: {
      dbType: "KeyValue",
    },
  },
});

// Ping Command
client.command({
  name: "ping",
  code: `Pong! $pingms`,
});

new PluginManager(bot).loadPlugins("default/comment"); //Loads the from the default folder ($comment function)
```

## Plugin Class

The Plugins class provides a way to load and manage plugins for your Discord bot.

To load plugins:
```javascript
pluginManager.loadPlugins(
    "default/os",
    "default/comment",
);
```

or

```javascript
pluginManager.load(); // if plugins are added via command line interface
```
